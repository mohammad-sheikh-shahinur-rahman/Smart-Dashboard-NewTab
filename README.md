### **Method 1: Chrome Web Store (Coming Soon)**
1. Visit the Chrome Web Store
2. Search for "Smart Dashboard NewTab"
3. Click "Add to Chrome"
4. Confirm installation

### **Method 2: Manual Installation (Current)**
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right toggle)
4. Click "Load unpacked" and select the `new-extension` folder
5. The extension will be installed and ready to use

### **Method 3: Production Package**
1. Download the production zip file from releases
2. Extract the zip file
3. Follow Method 2 steps 2-5

## üêõ **Troubleshooting**

### **Common Issues**

#### **Extension Not Loading**
- Check if Developer mode is enabled
- Verify the folder structure is correct
- Reload the extension from chrome://extensions/
- Check console for any error messages

#### **API Features Not Working**
- Verify API keys are correctly configured
- Check API key permissions and quotas
- Ensure internet connection is stable
- Check API configuration in settings

#### **Widgets Not Displaying**
- Check browser console for errors
- Verify all files are properly loaded
- Try refreshing the new tab page
- Check widget settings in the settings modal

#### **Performance Issues**
- Disable unnecessary widgets
- Clear browser cache and data
- Check for conflicting extensions
- Toggle visual effects off in settings

#### **Settings Not Working**
- Check if settings modal opens properly
- Verify all widget toggles are functional
- Check console for JavaScript errors
- Try resetting settings to default
